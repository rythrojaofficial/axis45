Swing King
Chain Challenge
  • 8m Warm-up Time
  • 3 attempts, with only the highest scoring pass counting
  • points awarded for each landed consecutive trick
  • Use any Transitions you like
  • .5 extra will be awarded for each additional trick utilized
    • e.g. aerial f/s raiz s/t gainer
    • 3 pts for 3 tricks + 2 pts for 2 extra tricks for 5 points
  • The Chain ends if:
    • Hands touch the floor any time after the initial setup
    • There is break in momentum (stopping)
    • More than two steps
    • 2 of the same vanish transition in a row
      • You can use tornado, but not 2 in a row
      • All other transitions are unlimited
